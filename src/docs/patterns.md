# Coding Patterns

## List of Coding Patterns

![List of Coding Patterns](https://i.ibb.co.com/qBLgs2N/algorithmic-patterns.png)

[_List of Coding Patterns_](https://github.com/DaudIbrahim/knowledge-base-pvt/blob/main/concepts/dsa/dsa-routine-learn.md#coding-patterns)

1. Two Pointers
   - Data Structure: Arrays, Strings, or Linked Lists.

2. Fast and Slow Pointers
   - Data Structure: Linked Lists, Arrays.
   - [NeetCode Linked List Cycle](https://youtu.be/gBTe7lFR3vc)

3. Sliding Window
   - Data Structure: Arrays, Strings, or Deques.
   - [Kantan Coding](https://youtube.com/playlist?list=PLxvbXPxg6ydxTJ99R8T-7DwPs2W1DuTVn)

4. Merge Intervals
   - Data Structure: Arrays (used for intervals), Sorting.
   - Familiarity with sorting algorithms like `quick sort` or `merge sort` is helpful for organizing intervals.

5. [Prefix Sum Array and Range Sum Queries](https://youtu.be/xbYr9JOC2Lk?si=imDCPi26TlxIPjKF)

6. In-place Reversal of Linked List
   - Data Structure: Linked Lists.

7. Two Heaps
   - Data Structure: Heaps (Min-Heap and Max-Heap).
   - Heap Data Structures

8. K-way Merge
   - Data Structure: Priority Queues (implemented using Heaps).
   - Priority Queues are typically implemented using Heap Data Structures

9. Top K Elements
   - Data Structure: Priority Queues (implemented using Heaps).
   - Priority Queues are typically implemented using Heap Data Structures

10. Modified Binary Search
    - Data Structure: Arrays.

13. Backtracking
    - Data Structure: Recursion Trees, Arrays, Strings, or Graphs.

    - Data Structure: Arrays, Matrices, or Hash Maps (for memoization).

15. Cyclic Sort
    - Data Structure: Arrays.

    - Data Structure: Graphs (Directed Acyclic Graphs, DAGs).

    - Data Structure: Matrices (2D Arrays).

20. Depth First Search
    - Data Structure: Graphs, Recursion (Stack for iterative approach).

21. Breadth First Search
    - Data Structure: Graphs, Queues.

22. Trie
    - Data Structure: Tries (Prefix Trees).

23. Hash Maps
    - Data Structure: Hash Maps (Dictionaries).

24. Knowing What to Track
    - Data Structure: Depends on the problem—often Hash Maps, Arrays, or Stacks.

25. Union Find
    - Data Structure: Disjoint Set Union (DSU), Arrays.

26. Custom Data Structures
    - Data Structure: Problem-specific, often involves designing custom classes/structures.

27. Bitwise Manipulation
    - Data Structure: Binary Representation (Numbers, Bits).
